## LAB 4 :

This is Mehak's JavaScript Quiz Game – a complete console-based interactive quiz running entirely in the browser. It uses a 6-question array storing trivia about geography, animals, and general knowledge, with answers normalized for case-insensitive matching via toLowerCase().trim().​

----------Core Functionality----------

runQuiz() loops through questions using prompt() for user input and alert() for immediate feedback (✅ Correct / ❌ Wrong with answer reveal)

Handles edge cases like canceled prompts (null) gracefully with skip messages

Calculates final percentage score and displays tiered feedback (Excellent ≥70%, Good ≥50%, Keep practicing)

Dual output via console.log() for debugging and browser console visibility

----------Technical Highlights----------

Array of objects for scalable question bank

String normalization prevents common input errors

Template literals for clean dynamic messages

Conditional feedback system with emoji enhancement

Self-contained – runs instantly when loaded

Perfect beginner JavaScript assignment demonstrating DOM interaction (prompt/alert), loops, conditionals, functions, and data structures in a fun, engaging format.
